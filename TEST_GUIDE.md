# 원신 파티 코스트 계산기 - 테스트 가이드

## 🚀 개발 서버 실행

### 터미널에서 다음 명령어 실행:

```bash
cd genshin-cost-calculator
npm run dev
```

**예상 출력:**
```
  VITE v7.3.1  ready in 123 ms

  ➜  Local:   http://localhost:5173/
  ➜  press h to show help
```

### 브라우저에서 열기:
- **로컬**: `http://localhost:5173/`

## 🧪 테스트 체크리스트

### 1️⃣ 초기 화면 확인
- [ ] 헤더에 "원신 파티 코스트 계산기" 제목 표시
- [ ] "전체 초기화" 버튼이 우측에 보임
- [ ] 파티 1과 파티 2가 각각 4개의 빈 슬롯으로 표시
- [ ] 총 코스트 패널에 "0" 표시

### 2️⃣ 파티 슬롯 클릭 (캐릭터 선택)
- [ ] 빈 슬롯(+)을 클릭
- [ ] 모달이 열림
- [ ] "캐릭터" 탭이 기본으로 활성화됨
- [ ] 캐릭터 그리드가 보임 (반응형 배치)

### 3️⃣ 캐릭터 선택
- [ ] 캐릭터 카드를 클릭 (예: "호두")
- [ ] 모달이 닫힘
- [ ] 슬롯에 선택된 캐릭터 정보 표시
  - 캐릭터 아이콘
  - 캐릭터 이름
  - 돌파 레벨 드롭다운 (0~6)
  - 코스트 표시

### 4️⃣ 무기 선택
- [ ] 슬롯의 무기 영역 클릭 (또는 다른 빈 슬롯의 무기 부분)
- [ ] 모달이 열림
- [ ] **무기** 탭으로 변경됨
- [ ] 무기 그리드가 보임
- [ ] 무기 카드를 클릭 (예: "원시옥 칼")
- [ ] 무기 정보가 슬롯에 업데이트됨
  - 무기 아이콘
  - 무기 이름
  - 무기 돌파 드롭다운 (0~5)

### 5️⃣ 돌파 레벨 변경
- [ ] 캐릭터 돌파 드롭다운 변경 (0→5)
- [ ] **코스트가 자동 재계산됨** (0 → 6000)
- [ ] 무기 돌파 드롭다운 변경 (0→3)
- [ ] **코스트가 자동 재계산됨** (6000 → 6750)

### 6️⃣ 코스트 계산 검증
**테스트 값:**
- 캐릭터 돌파 5 = 6000
- 무기 돌파 3 = 750
- 슬롯 코스트 = 6750

**확인:**
- [ ] 슬롯 카드에 "코스트: 6,750" 표시
- [ ] 파티 1 코스트 합계 업데이트됨
- [ ] 총 코스트 패널 업데이트됨

### 7️⃣ 슬롯 초기화
- [ ] 슬롯의 "초기화" 버튼 클릭
- [ ] 확인 대화 표시됨
- [ ] 확인 클릭
- [ ] 슬롯이 비워짐 (+ 표시로 돌아옴)
- [ ] 코스트가 0으로 초기화됨

### 8️⃣ 중복 선택 방지
- [ ] 파티 1 슬롯 1에서 "호두" 선택
- [ ] 다른 슬롯의 모달에서 캐릭터 탭 확인
- [ ] "호두" 카드가 **흐려짐 (disabled 상태)**
- [ ] "호두" 카드 클릭 불가능

### 9️⃣ 반응형 테스트
#### 데스크톱 (1200px+)
- [ ] 파티 슬롯이 4열 그리드로 표시
- [ ] 아이템 그리드가 5열로 표시

#### 태블릿 (768px~1199px)
- [ ] 파티 슬롯이 2열 그리드로 표시
- [ ] 아이템 그리드가 4열로 표시

#### 모바일 (768px 이하)
- [ ] 파티 슬롯이 1열로 표시
- [ ] 아이템 그리드가 3열로 표시
- [ ] 모달이 전체 화면 차지

### 🔟 전체 초기화
- [ ] 여러 캐릭터/무기 선택 후 "전체 초기화" 클릭
- [ ] 확인 대화 표시됨
- [ ] 확인 클릭
- [ ] **모든 슬롯이 초기화됨**
- [ ] 총 코스트가 0으로 리셋됨

## 🐛 주요 테스트 케이스

### 케이스 1: 풀 파티 구성
```
파티 1:
- 슬롯 1: 호두 (돌파 6) + 원시옥 칼 (돌파 5) = 10000 + 3000 = 13000
- 슬롯 2: 감우 (돌파 4) + 아모스의 활 (돌파 3) = 3000 + 750 = 3750
- 슬롯 3: 벤티 (돌파 2) + 세실의 단검 (돌파 2) = 500 + 250 = 750
- 슬롯 4: 진 (돌파 1) + 무기 미선택 = 100 + 0 = 100

파티 1 합계 = 17600

파티 2: (비어있음)
파티 2 합계 = 0

**총 코스트 = 17600**
```

### 케이스 2: 일부 슬롯만 채움
```
파티 1:
- 슬롯 1: 호두 (돌파 3) + 원시옥 칼 (돌파 2) = 1500 + 250 = 1750
- 슬롯 2~4: 비어있음

파티 1 합계 = 1750
```

## 🎮 UI 상호작용 가이드

### 탭 전환
- 모달 상단의 "캐릭터" 또는 "무기" 탭 버튼 클릭
- 탭이 전환되고 그리드가 업데이트됨

### 검색 (차후 추가 예정)
- 현재 구현되지 않음
- 향후 버전에서 추가 예정

### 모달 닫기
- 아이템 선택 시 자동 닫힘
- 우상단 "×" 버튼 클릭
- 오버레이 (어두운 배경) 클릭

## ⚠️ 알려진 제약사항

1. **아이콘 이미지**: 실제 원신 이미지가 아님 (플레이스홀더 경로)
   - `/data/characters/{id}.png` 형식으로 참조
   - 실제 이미지가 없으면 브라우저 기본 이미지 오류 표시

2. **검색 기능**: 미구현 (차후 추가 예정)

3. **저장/불러오기**: 미구현 (로컬스토리지 사용 가능)

4. **색상 테마**: 라이트 모드만 지원 (다크모드는 차후 추가)

## 🔍 문제 발생 시

### 서버가 시작되지 않음
```bash
# 포트 이미 사용 중인 경우
npm run dev -- --port 3000
```

### 데이터 로딩 실패
- `public/data/` 폴더에 JSON 파일이 있는지 확인
- 브라우저 DevTools Console 탭에서 오류 메시지 확인

### 스타일이 적용되지 않음
- `npm run dev` 재시작
- 브라우저 캐시 초기화 (`Ctrl + Shift + Delete`)
